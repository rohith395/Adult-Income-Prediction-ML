# Adult-Income-Prediction-ML
Machine Learning project to predict whether a person's income exceeds $50K using the Adult Census dataset.
Adult Census dataset from UCI Machine Learning Repository.

## Workflow
1. Data cleaning
2. Exploratory Data Analysis
3. Feature engineering
4. SMOTE for class imbalance
5. Feature scaling
6. Model training
7. Model comparison
8. Hyperparameter tuning
9. Model evaluation

## Models Used
- Logistic Regression
- Random Forest
- Gradient Boosting
- Support Vector Machine
- KNN

## Best Model
Gradient Boosting achieved the best F1 score.

## Tools
Python, Pandas, Scikit-Learn, Google Colab
